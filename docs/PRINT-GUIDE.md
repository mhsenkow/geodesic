# Print Guide

Recommendations for printing geodesic hub connectors generated by this tool.

## General settings

| Setting | PLA/PETG | ABS/ASA |
|---------|----------|---------|
| Layer height | 0.2 mm | 0.2 mm |
| Wall/perimeters | ≥4 | ≥5 |
| Infill | 30–50% gyroid | 40–60% gyroid |
| Top/bottom layers | 5+ | 6+ |
| Supports | Usually none (foot on bed) | Usually none |

## Orientation

- Use the **Hub Inspector** build plate guide — cyan grid = bed surface
- Default auto-orientation aligns print-up opposite the average socket direction
- Override with **Print-Up Vector** for difficult hubs (e.g. base ring hubs)
- Print with the build foot on the bed when enabled
- Tune **Base Thickness** and **Build Foot Size** when printing warp-prone materials or tall hub styles

## Tolerance tuning

1. Pick your **Standard Stock Size** from the catalog (2×4, 3/4" PVC, 1" EMT, etc.) — actual mm dimensions are applied automatically.
2. Print a single test hub at default tolerance
3. Insert your strut (PVC, EMT, or lumber)
4. If too tight: increase **Printer Tolerance** by 0.05 mm
5. If too loose: decrease by 0.05 mm
6. Re-export STL after changes

The inspector reports the generated socket opening and socket depth for the selected hub, so you can check whether the modeled clearance matches your measured stock before exporting a full batch.

### Common stock reference (actual sizes)

| Trade name | Actual cross-section |
|------------|---------------------|
| 1×4 lumber | 19 × 89 mm |
| 2×4 lumber | 38 × 89 mm |
| 2×6 lumber | 38 × 140 mm |
| 3/4" PVC Sch40 | OD 26.7 mm |
| 1" PVC Sch40 | OD 33.4 mm |
| 1" EMT conduit | OD 29.5 mm |
| 1/2" solid rod | Ø 12.7 mm |

## Screw pilot holes

Enable **Screw Pilot Holes** in the Material panel:

- **Timber hubs:** through-holes in socket walls for #8 or M5 wood screws into the lumber edge
- **Round tube hubs:** set-screw dimples in the socket wall (drill/tap or use grub screws)

Use 4.2 mm (#8) for 2×4 framing, 5.0 mm for heavy 2×6 connections, 3.3 mm (M4) for EMT set screws.

### By material

- **PLA**: 0.25–0.35 mm shrinkage offset typical
- **PETG**: 0.30–0.40 mm
- **ABS/ASA**: 0.35–0.50 mm (higher shrinkage)
- **Timber**: 0.35–0.45 mm — wood swells slightly in humidity

## Wall thickness

- Minimum **3 mm** for small PVC hubs (V1–V2)
- **5 mm** default works well for structural domes
- Timber hubs benefit from **5–6 mm** walls due to higher load

## Mesh detail

- Preview uses low detail for speed
- Export uses ≥64 segments for smooth sockets
- Higher detail = larger STL files and longer slice times; 48–64 is usually sufficient

## Assembly tips

1. Export **Strut Lengths CSV** and cut all struts before printing hubs
2. Use **Hub Map SVG** to label each vertex position during assembly
3. Export the **Test Set ZIP** and print one of each unique hub type first
4. Base hubs often need the build foot — do not remove unless using a brim in slicer
5. Use the **Production Set ZIP** or **Build Plate 3MF** only after the test hubs fit your stock

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Strut won't insert | Increase tolerance 0.05 mm |
| Hub cracks at socket | Increase wall thickness or infill |
| Foot delaminating | Enable brim in slicer, reduce foot margin |
| STL warnings on export | Increase mesh detail; check for non-manifold geometry in slicer |
