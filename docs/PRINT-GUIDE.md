# Print Guide

Recommendations for printing geodesic hub connectors generated by this tool.

## General settings

| Setting | PLA/PETG | ABS/ASA |
|---------|----------|---------|
| Layer height | 0.2 mm | 0.2 mm |
| Wall/perimeters | ≥4 | ≥5 |
| Infill | 30–50% gyroid | 40–60% gyroid |
| Top/bottom layers | 5+ | 6+ |
| Supports | Usually none (foot on bed) | Usually none |

## Orientation

- Use the **Hub Inspector** build plate guide — cyan grid = bed surface
- Default auto-orientation aligns print-up opposite the average socket direction
- Override with **Print-Up Vector** for difficult hubs (e.g. base ring hubs)
- Print with the build foot on the bed when enabled

## Tolerance tuning

1. Print a single test hub at default tolerance (0.3 mm)
2. Insert your strut (PVC or timber)
3. If too tight: increase **Printer Tolerance** by 0.05 mm
4. If too loose: decrease by 0.05 mm
5. Re-export STL after changes — tolerance is baked into socket geometry

### By material

- **PLA**: 0.25–0.35 mm shrinkage offset typical
- **PETG**: 0.30–0.40 mm
- **ABS/ASA**: 0.35–0.50 mm (higher shrinkage)
- **Timber**: 0.35–0.45 mm — wood swells slightly in humidity

## Wall thickness

- Minimum **3 mm** for small PVC hubs (V1–V2)
- **5 mm** default works well for structural domes
- Timber hubs benefit from **5–6 mm** walls due to higher load

## Mesh detail

- Preview uses low detail for speed
- Export uses ≥64 segments for smooth sockets
- Higher detail = larger STL files and longer slice times; 48–64 is usually sufficient

## Assembly tips

1. Export **Strut Lengths CSV** and cut all struts before printing hubs
2. Use **Hub Map SVG** to label each vertex position during assembly
3. Print one of each unique hub type first (batch export)
4. Base hubs often need the build foot — do not remove unless using a brim in slicer

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Strut won't insert | Increase tolerance 0.05 mm |
| Hub cracks at socket | Increase wall thickness or infill |
| Foot delaminating | Enable brim in slicer, reduce foot margin |
| STL warnings on export | Increase mesh detail; check for non-manifold geometry in slicer |
