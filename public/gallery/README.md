# Example Gallery

Share photos of printed hubs and assembled domes here.

## How to contribute an example

1. Print hubs using a named preset (e.g. Greenhouse 4m V2)
2. Note exact settings: V-level, diameter, material, tolerance, filament type
3. Add a photo to this folder: `public/gallery/your-name-dome.jpg`
4. Add an entry below
